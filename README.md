# XML Based Data Serialization Project with Custom Static Libraries (`ezxmlfihrist`)

This project focuses on data serialization and parsing techniques in C/C++ by integrating a third-party XML library alongside a custom-built static library.

## 🚀 Features
* **Custom Static Library Integration:** Reused core directory algorithms by linking a custom Static Library (`staticlibfihrist`) into the XML parser ecosystem.
* **Third-Party Integration:** Successfully integrated and managed the open-source `ezxml` library within a C/C++ environment.
* **Data Serialization:** Implements serialization and deserialization processes to convert runtime data into structured XML files and vice versa.
* **Secure Input & Parsing:** Designed to safely parse external structured files, building a foundational understanding of secure input validation.

## 🛠️ Tech Stack
* C / C++
* Static Libraries (`.lib`)
* XML Parsing & Data Serialization
* `ezxml` Library
* Visual Studio 2022
